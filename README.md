H_InventoryUtils é um plugin para Spigot/Paper que fornece utilitários avançados para manipulação de inventários no Minecraft. Ele oferece funções para gerenciamento de itens, empilhamento automático, controle de interação e muito mais.

Recursos
Empilhamento automático de itens até um limite configurável
Verificação de mudanças no inventário
Controle avançado de interação com inventários
Suporte para eventos personalizados de inventário
Configuração flexível via config.yml
